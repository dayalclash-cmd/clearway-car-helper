# Clearway Car Hire

Independent car hire consultancy in Ireland. Get honest advice, clear pricing, and personal service.

**Live site:** [clearwaycarhire.ie](https://clearwaycarhire.ie)

## Tech Stack

- **Framework:** React 18 + TypeScript
- **Build:** Vite 5
- **Styling:** Tailwind CSS + shadcn/ui
- **Hosting:** Cloudflare Pages

## Getting Started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

## Deployment

This project deploys to **Cloudflare Pages**:

- **Build command:** `npm run build`
- **Output directory:** `dist`
